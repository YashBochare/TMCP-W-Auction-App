# Goals and Background Context

## Goals
- Deliver a live, highly engaging, semi-controlled digital auction experience that perfectly mimics real-world sports auctions.
- Guarantee 100% mathematical accuracy enforcing the 100,000 virtual purse limit and exactly 7-player squad constraint for all Captains.
- Provide ultra-low latency real-time synchronization (<500ms) across Auctioneer, Captain, and Viewer interfaces.
- Empower a human Auctioneer with absolute control over the bidding pace, bid acceptance, and edge-case resolution (undo, manual overrides).
- Create a reusable, easily configurable platform for future club events (variable team counts, Excel-based player ingestion).

## Background Context
The Toastmasters club hosts an "Impromptu Premier League" event where captains build teams by bidding on players. Traditional automated auction systems remove the human drama and pacing required for a live event, while manual tracking (spreadsheets/whiteboards) is slow, lacks transparency, and limits audience engagement. This system bridges the gap by offering a premium, sports-broadcast visual experience for the audience, mobile bidding for captains, and a centralized manual control dashboard for the auctioneer. The application will elevate the event's production value and streamline squad-building logistics.
